  1. Що таке файл package.json?

    2. package.json - це файл, який містить дані про проект та залежності, необхідні для його розробки та запуску

  2. Які є типи запитів HTTP?
    3. Основні - GET, POST, PUT, DELETE, PATCH. Запити підтримують query параметри для більш детальної обробки
    • GET - отримання данних
    • POST - відправка даних на сервер для запису
    • PUT,PATCH - зміна данних на сервері , запит включає параметр id, body - данні для заміни (example url - 'http://localhost:3000/user/:id' - :id = userId)
    • DELETE - видалення данних, запит включає параметр id (example url - 'http://localhost:3000/user/:id')
  3. Поясніть різницю між локальним і глобальним встановленням пакетів npm
    • Локальне встановлення пакетів означає, що пакети встановлюються в конкретну директорію вашого проекту, і доступні лише для цього проекту.
    • Глобальне встановлення пакетів означає, що пакети встановлюються на рівні вашої системи, і доступні для використання будь-де на вашому комп'ютері.
